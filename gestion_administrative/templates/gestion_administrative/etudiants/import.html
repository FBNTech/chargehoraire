Importer des étudiants depuis Excel

Instructions d'importation

Votre fichier Excel doit contenir les colonnes suivantes :

  • Colonnes obligatoires : matricule, nom_complet
  • Colonnes optionnelles : date_naissance, sexe, telephone, departement, classe, annee_academique

Note : Les étudiants existants (même matricule) seront mis à jour avec les nouvelles données. Le format de date doit être reconnaissable par Excel (ex: 01/01/2000).

Exemple de format Excel
matricule nom_complet date_naissance sexe telephone departement classe
ET001 DUPONT Jean 01/01/2000 M 123456789 Informatique L3 Informatique
ET002 MARTIN Marie 15/03/1999 F 987654321 Gestion M1 Gestion
